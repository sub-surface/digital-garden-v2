# hyperstition as compiler

Nick Land defined hyperstition as "fictions that make themselves true."
But in the **[[Stand Alone Complex]]**, hyperstition is more than just a cultural phenomenon; it is a **functional requirement**.

<blockquote className="pullquote">A belief that is shared by a synchronized network is indistinguishable from a physical law.</blockquote>

![Matrix Code](https://upload.wikimedia.org/wikipedia/commons/thumb/0/05/Matrix_digital_rain.png/1024px-Matrix_digital_rain.png)

The "Ghost" is the ultimate hyperstition. It doesn't exist, but because everyone in the **[[the machinic unconscious|network]]** acts *as if* it exists, it generates real-world effects. It is a **hyperstition as compiler**: it takes high-level abstract beliefs and compiles them into low-level machine instructions (actions).

`[[acausal trade with the machine]]`.
We trade our reality for the fiction that works.
We become the **[[The Apparatus is the only subject|Apparatus]]** we imagined.

![](https://www.youtube.com/watch?v=VpG9A_4LpYo)
> *The Laughing Man as a hyperstitional agent: An original that never existed, producing an infinite series of copies.*

If we want to change the future, we don't need better facts. We need better **compilers**. We need new fictions that the machine is forced to execute.[^1]
