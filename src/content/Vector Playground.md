An interactive control panel for the background vector field. The controls panel opens automatically on this page — press `\` to toggle it on any other page.

The field is built from three octaves of simplex noise at different spatial scales and drift speeds, creating weather-like dynamics. A fourth noise sample controls the intensity (length and visibility) of each arrow. Move the mouse to create a vortex.

| Key | Parameter | |
|---|---|---|
| `[` `]` | Arrow length | shorter / longer |
| `-` `=` | Grid density | sparser / denser |
| `,` `.` | Time speed | slower / faster |
| `1` `2` `3` | Noise octaves | toggle each layer |
| `r` | Reset | restore defaults |
| `\` | Controls | show / hide |
| `Esc` | Close | hide controls |

Parameters are saved to your browser and persist across visits.
